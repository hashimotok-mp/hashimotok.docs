# ロールパネル
## リアクション式
### 単独追加(add)
指定したメッセージのリアクションに対して、追加用のロールを設定します。
`?rr add [チャンネルID] <メッセージID> <リアクション> <ロール>`
### 単独削除(remove)
指定したメッセージのリアクションに対して、削除用のロールを設定します。
`?rr remove [チャンネルID] <メッセージID> <リアクション> <ロール>`
### 一括追加(addmany)
一括で指定したメッセージのリアクションに対して，追加用のロールを設定します。
`?rr addmany [チャンネルID] <メッセージID> <リアクション> <ロール> ...`
### 一括削除(removemany)
一括で指定したメッセージのリアクションに対して，削除用のロールを設定します。
`?rr removemany [チャンネルID] <メッセージID> <リアクション> <ロール> ...`
### ホワイトリスト
いずれかのロールを持っている場合、リアクション式のロールパネルが機能するようになります。
`?rr whitelist [チャンネルID] <メッセージID> <ロール>`
### ブラックリスト
いずれかのロールを持っている場合、リアクション式のロールパネルが機能しなくなります。
`?rr blacklist [チャンネルID] <メッセージID> <ロール>`
### 全ホワイトリスト
全てのロールを持っている場合、リアクション式のロールパネルが機能するようになります。
`?rr whitelistall [チャンネルID] <メッセージID> <ロール>`
### 全ブラックリスト
全てのロールを持っている場合、リアクション式のロールパネルが機能しなくなります。
`?rr blacklistall [チャンネルID] <メッセージID> <ロール>`

# ロールパネル設定
## リアクション式
### 重複禁止
他のロールを持っている場合、リアクション式のロールパネルでロールを付与できなくなります。
`?rr unique [チャンネルID] <メッセージID>`
### 複数選択
複数選択が可能になります。
`?rr normal [チャンネルID] <メッセージID>`
### 重複禁止切り替え
他のロールを持っている場合、他のロールを外して新しいロールを付与するようになります。
追加と削除を設定している場合は，他のロールは追加と削除が反転して機能します。
`?rr toggle [チャンネルID] <メッセージID>`

# 期間限定
## リアクション式
### 追加
`?rr addtemp [チャンネルID] <メッセージID> <リアクション> <ロール> <期間>`
### 削除
`?rr removetemp [チャンネルID] <メッセージID> <リアクション> <ロール>`
### 期間の書式
`1d` (1日), `2h` (2時間), `30m` (30分) など
